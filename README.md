# Wasabi — Releases

Public downloads for **Wasabi**, a native, low-RAM macOS shell that hosts pinned
web chat apps (WhatsApp Web, Telegram WebK) — each in its own `WKWebView`, behind
a vertical icon sidebar. A single native app on the system WebKit in place of an
Electron multi-chat client, at a fraction of the RAM.

## Download

Grab the latest signed, notarized `Wasabi.dmg` from the
[**Releases**](https://github.com/gin-so/wasabi-releases/releases) page, open it,
and drag Wasabi to Applications.

- macOS 15+ (Apple Silicon)
- Signed with a Developer ID and notarized by Apple — no Gatekeeper warning.

> This repository hosts release binaries only. Source lives in a separate private
> repository.
